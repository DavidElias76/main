# 🚀 Full Stack Development Journey

I began with the foundations of web development — **HTML, CSS, and JavaScript** — then advanced into **React for frontend development** and later expanded into **backend development with Node.js, Express, databases, APIs, and authentication systems**.

The goal of this repository is to showcase my continuous growth as a developer through practical projects, clean architecture, and modern development practices.
